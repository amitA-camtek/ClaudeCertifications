# 12-Week Claude Certified Architect — Foundations Prep

Source: [cilbus.txt](../cilbus.txt) (original 7-day/week program, compressed here to 2 days/week).

## Study rules

- **2 study days per week**, 2.5 hours each = **5 h / week**
- **Study Day** — theory + small build (original Days 1–5 content)
- **Practice Day** — finish build + 10-question practice test + wrong-answer review (original Day 6)
- Original Day 7 rest is dropped (5 non-study days already exist)

## Session templates

**Study Day (2.5h)**
- 0:00–0:10 — warmup: re-read last week's wrong answers
- 0:10–1:40 — theory reading + notes
- 1:40–2:20 — hands-on sketch / build
- 2:20–2:30 — write 3-bullet summary to `notes/`

**Practice Day (2.5h)**
- 0:00–0:30 — finish/polish week's build
- 0:30–1:15 — Practice Test (10 Q)
- 1:15–2:00 — review wrong answers, log in `practice_test/`
- 2:00–2:30 — update `notes/weak_spots.md` (feeds W12 targeted review)
